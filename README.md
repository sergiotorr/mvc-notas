# mvc-notas

Aplicación web simple desarrollada con Node.js, Express y EJS para practicar el patrón MVC.

## Tecnologías
- Node.js
- Express
- EJS

## Estructura

models/
Contiene el modelo de datos (notas).

controllers/
Controladores que manejan las rutas y lógica de negocio.

views/
Vistas EJS que muestran el listado y formulario.

## Ejecutar

Instalar dependencias:

npm install

Ejecutar aplicación:

node app.js

Abrir en navegador:

http://localhost:3000
